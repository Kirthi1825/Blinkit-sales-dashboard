# Blinkit Sales Dashboard

## Project Overview

This Power BI dashboard analyzes Blinkit's grocery sales data and provides business insights related to sales performance, customer behavior, product categories, and outlet operations.

## Dashboard Pages

### 1. Sales Overview
- Total Sales
- Average Sales
- Number of Items
- Average Rating
- Outlet Establishment Trends
- Fat Content Analysis
- Item Type Performance

### 2. Product Sales Analysis
- Top 10 Products
- Bottom 10 Products
- Category Contribution
- Product Revenue Analysis
- Business Recommendations

### 3. Customer & Operational Insights
- Item Visibility Analysis
- Revenue by Outlet
- Outlet Performance
- Rating Distribution
- Operational Recommendations

## Tools Used

- Power BI
- Power Query
- DAX
- Excel/CSV Dataset

## Key Insights

- Fruits & Vegetables generated the highest sales.
- Tier 3 outlets contributed the highest revenue.
- Average product rating remained around 4.
- High-rated categories showed strong growth potential.
